## 📌 Project Overview

Healthcare organizations generate massive amounts of patient data every day. However, raw data alone cannot support decision-making until it is properly cleaned, analyzed, and visualized.

This project focuses on performing a complete **Exploratory Data Analysis (EDA)** on a healthcare dataset to discover meaningful trends, hidden patterns, relationships between variables, and insights that can support data-driven healthcare decisions.

The project follows a real-world data analytics workflow:

- Data Understanding
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- Statistical Analysis
- Hypothesis Testing
- Business Insights
- Final Conclusions

---

# 🎯 Project Objectives

The main objective of this project is to answer important healthcare-related questions using data analysis.

Some of the key objectives include:

- Analyze patient demographics
- Understand medical condition distributions
- Study billing patterns
- Identify high-cost patients
- Explore admission trends
- Investigate relationships between age and disease severity
- Analyze medication usage
- Study insurance provider distributions
- Generate actionable healthcare insights

# 📖 Hypothesis Testing

One of the major hypotheses explored in this project:

> **Hypothesis:** Older patients have a higher chance of experiencing more severe or abnormal medical outcomes than younger patients.

The hypothesis was evaluated using statistical analysis and visual evidence to support data-driven conclusions.

# 💡 Key Insights

Some important findings from the analysis include:

- Patient demographics reveal significant variation across age groups.
- Medical conditions are not equally distributed.
- Billing amounts vary considerably across admission types.
